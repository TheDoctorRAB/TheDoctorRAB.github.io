# NE585: Nuclear fuel cycle analysis ## Project 5 ### Reactor operation ####[400] <p> 1. [100] Let's do some depletion analysis. Solve the Pu-239 production equation. $$\frac{dN_{49}}{d\theta}=N_{28}^0\sigma_{28}+\kappa_{25} N_{25} \sigma_{25}-\gamma_{49} N_{49} \sigma_{49}+\kappa_{41} N_{41} \sigma_{41}$$ $$N_{49}(0)=0$$ $$\kappa_m=\eta_m \epsilon P_F (1-p)+\eta_m \frac{\alpha_{28}}{1+\alpha_{28}}\cdot\frac{\epsilon-1}{\eta_{28}-1}$$ $$\gamma_m=1-\kappa_m$$ Just to get the ball rolling, the solution is as follows: $$N_{49}(\theta)=C_1(1-e^{-\sigma_{49}\gamma_{49}\theta})+C_2(e^{-\sigma_{25}\theta}-e^{-\sigma_{49}\gamma_{49}\theta})$$ $$C_1=\frac{N_{28}^0\sigma_{28}}{\sigma_{49}\gamma_{49}},C_2=\frac{\kappa_{25} N_{25}^0 \sigma_{25}}{\sigma_{49}\gamma_{49}-\sigma_{25}}$$ 2. [50] Solve for Pu-240, and the rest of the Pu isotopes in the same way. $$\frac{dN_{40}}{d\theta}=\frac{\alpha_{49}}{1+\alpha_{49}}N_{49}\sigma_{49}-N_{40}\sigma_{40}$$ 3. [50] Solve for Pu-241. $$\frac{dN_{41}}{d\theta}=N_{40}\sigma_{40}-N_{41}\sigma_{41}$$ 4. [50] Solve for Pu-242. $$\frac{dN_{42}}{d\theta}=\frac{\alpha_{41}}{1+\alpha_{41}}N_{41}\sigma{41}-N_{42}\sigma_{42}$$ For 2-4: $$N_m(0)=0$$ 5.
